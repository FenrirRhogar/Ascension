# ⚔️ Ascension: Split-Screen Co-op

<p align="center">
  <b>A high-intensity local multiplayer 3D action-adventure game.</b><br>
  <i>Developed by Ioannis Bouritis for Graphics Unity Project 2026</i>
</p>

## 📜 Overview
**Split-Screen Co-op Ascension** drops 2 to 4 players into a hostile, procedurally generated environment where movement, resource management, and combat are shared responsibilities. Work together to survive dynamic biomes, defeat waves of enemies, and reach the level gate to ascend! 

## ✨ Key Features
- **Team Synergy:** Success depends entirely on players watching each other's backs.
- **Dynamic Split-Screen:** Viewports automatically resize based on the number of active players.
- **Procedural Challenge:** Custom noise algorithms and splat-mapped materials create dynamic ground meshes, meaning no two runs are identical!
- **Atmospheric Immersion:** 5 distinct biomes (Forest, Volcanic, Frozen, Desert, Autumn) with dynamic lighting, linear fog, and a random "Night Mode".

## 🧙‍♂️ Classes
Players can select from four distinct classes, each with unique abilities and playstyles:
* 🛡️ **Fighter:** High health, melee-focused tank.
* 🔮 **Wizard:** Ranged fireballs and area-of-effect spells.
* 🗡️ **Assassin:** High speed, high critical damage.
* 💖 **Cleric:** Healing and support abilities.

## 🎮 Controls
The game supports concurrent input from multiple players using Keyboard & Mouse (KBM) or Universal Gamepads.

| Action | Player 1 (KBM) | Player 2 (IJKL) | Gamepad |
| :--- | :--- | :--- | :--- |
| **Move** | W / A / S / D | I / J / K / L | Left Stick |
| **Look / Aim** | Mouse Delta | Numpad 8, 4, 5, 6 | Right Stick |
| **Attack** | Left Mouse | Numpad 7 | Right Trigger |
| **Ability** | Right Mouse | Numpad 9 | Left Trigger |
| **Ultimate** | Q | U | North Button (Y) |
| **Consumable** | 1, 2, 3 | Numpad 7, 8, 9 | D-Pad (L/U/R) |
| **Pick Up** | F | Semicolon (;) | West Button (X) |
| **Toggle POV** | C | Y | R3 (Stick Click) |

## 🎒 Loot Economy
During your ascension, you will find items dropped by defeated enemies:
* 🧪 **Potions (Consumables):** Provide immediate restoration (Health, Mana, Stamina) or temporary buffs (Speed Booster, Ultimate Potion).
* 💎 **Artifacts (Passive Buffs):** Permanent run-wide stat boosts like the *Amulet of Life* (+Max Health) or *Sword of Might* (+15% Damage).
